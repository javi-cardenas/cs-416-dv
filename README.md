# cs-416-dv

Final Project Narrative Visualization

## Setup

Install Node.js using Node Version Manager (NVM)...

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

Within the project directory...

```sh
nvm install
nvm use
```

To run the project...

```sh
npx live-server
```
